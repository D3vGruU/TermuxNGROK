# TermuxNGROK

This a Script installs the NGROK to the Termux and Make NGROK to run flawlessly......

# Getting Started

#TO RUN

 git clone https://github.com/D3vGruU/TermuxNGROK.git
 
 cd TermuxNGROK
 
 chmod 777 TNGROK.sh
 
 sh TNGROK.sh
 
 ./ngrok authtoken (ur id)
